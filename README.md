## 📋 О проекте

Данный репозиторий содержит исходный код официального сайта-визитки Черноусова Антона Анатольевича. Сайт создан с использованием современных технологий веб-разработки и предоставляет следующую информацию.

1. Личные данные
2. О себе (Professional Summary / About)
3. Опыт работы
4. Образование
5. Навыки (Skills)
6. Портфолио / Проекты
7. Сертификаты и курсы
8. Дополнительная информация

**Особенности:**

- Адаптивный дизайн (работает на любых устройствах)
- Высокая производительность и скорость загрузки
- Современный UI/UX
- Полная интеграция с поисковыми системами (SEO)

---

## ⚙️ Технологический стек

- **Framework**: Vue.js 3 (Composition API)
- **Routing**: Vue Router
- **State Management**: Pinia
- **Build Tool**: Vite
- **Styling**: SCSS, Flexbox, CSS Grid
- **Deployment**: GitHub Pages / Vercel / Netlify

## 🚀 Запуск проекта локально

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/anton-c/anton-c.ru.git
cd anton-c.ru
```

2. **Установите зависимости:**

```bash
npm install
# или
pnpm install
```

3. **Запустите в режиме разработки:**

```bash
npm run dev
```

4. **Откройте в браузере:**

```
http://localhost:5173
```

## 🔨 Сборка для продакшена
